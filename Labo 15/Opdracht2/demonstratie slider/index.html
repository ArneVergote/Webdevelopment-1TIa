Een gekleurd blokje :

Waar wordt de event listener gekoppeld aan de slider?

De event listener wordt gekoppeld aan de eerste slider (sliders[0]) met addEventListener("input", update) en addEventListener("change", update).

Waarom moeten we die op twee soorten events koppelen?

Het input-event zorgt voor realtime updates tijdens het slepen, terwijl change triggert als de waarde is ingesteld.

In de CSS file wordt nergens een rode kleur opgegeven, waar wordt dan wel de rode kleur van het blokje ingesteld?

De rode kleur wordt in JavaScript ingesteld via colorDemos[0].style.backgroundColor = "red";

Waarom schrijven we telkens sliders[0] en colorDemos[0] en niet gewoon sliders of colorDemos?

Omdat sliders en colorDemos HTMLCollections zijn, gebruiken we [0] om het eerste element te selecteren.

Waar vind je de CSS-properties die programmatorisch werden ingesteld, zoals bv. de rode achtergrondkleur van het blokje?

De CSS-properties die programmatorisch zijn ingesteld, staan in element.style
